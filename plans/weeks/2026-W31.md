# Week 2026-W31 (Jul 27–Aug 2) — Base Phase, Week 7

## Context

Two firsts this week:

1. **First brick session** (Sat) — the Saturday long ride gets a short easy run bolted on
   immediately after. Running off the bike is a race-specific skill; we start it now and
   build it through Build/Peak (see macro *Run & Brick Strategy*).
2. **Last beach volleyball** (Wed ~Jul 29). From next week Wednesday evenings are free and
   become the 3rd swim slot.

This is the **peak load week** of the current mini-block — next week (W32) is a planned
**deload**. Push a little here knowing recovery is scheduled.

**Zones** (FTP 238W, tested Jul 3): Bike by **power**. Run by HR + pace.
- Bike Z2 = 131–178W (56–75%) · Z3 tempo = 179–214W (76–90%)
- Run Z2 = 125–154 bpm · Z3 tempo = 155–169 bpm (~4:18–4:48/km)

### Sat Aug 1
**First Brick** — Long Ride + short transition run

- **Ride**: Z2 endurance (**131–178W**, ~150–170W), ~2:30h, ~70–80km, real elevation if
  possible — let watts rise into Z3 on climbs but keep the flats in Z2. Fuel 60–80g
  carbs/h from 45min.
- **Immediately off the bike** (change shoes fast, <5min): **15min easy run**. Legs will
  feel strange/heavy — that's the point. Keep it easy Z2, short strides of a stride if it
  loosens up. Don't chase pace.

First brick — keep the run deliberately short. We build duration over the coming months.

---

### Sun Aug 2

Legs carry brick fatigue from Saturday. Easy Z2 (125–154 bpm), run by feel. Shorten to
